# 🌾 Smart Farming Assistant
Smart Farming Assistant is a Machine Learning based web application that predicts suitable crops using soil nutrients and climate conditions like NPK, temperature, humidity, pH, and rainfall. It also recommends fertilizers and provides farming tips through an interactive Streamlit - based user interface.”

Smart Farming Assistant is a Machine Learning based web application developed to help farmers make smart agricultural decisions. The system predicts the most suitable crop based on soil nutrients and environmental conditions such as Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, pH, and rainfall. It also recommends fertilizers and provides useful farming tips for better crop growth and productivity.
